# 🏀 Bounce

### AI-Generated Physics Demo

I challenged **Gemini** to write a simulation of a ball bouncing down a ramp, complete with realistic physics effects, just to see how capable the model actually is at generating functional code.

> **The Verdict:** I must say, I’m quite impressed with the results.

### 🛠️ Project Overview

* **Core Mechanic:** Dynamic ball physics on an inclined plane.
* **Key Features:** Gravity simulation, momentum, and collision detection.
* **Engine:** Purely AI-generated logic.
